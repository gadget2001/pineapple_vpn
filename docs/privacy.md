# Политика конфиденциальности Pineapple VPN

Дата: 2026-03-11

## 1. Какие данные собираются
- Telegram ID и username
- IP-адрес подключения
- Время подключения
- Данные о подписках и платежах

## 2. Цели обработки
Данные используются для предоставления сервиса, учета подписок, поддержки пользователей и соблюдения юридических требований.

## 3. Хранение данных
Логи подключений хранятся 30 дней. Данные подписок и платежей хранятся в течение срока, необходимого для исполнения обязательств и выполнения требований законодательства.

## 4. Передача третьим лицам
Передача данных возможна только по запросу уполномоченных органов или для исполнения платежей (ЮKassa).

## 5. Безопасность
Применяются меры защиты: ограничение доступа, контроль запросов, шифрование и аудит.
