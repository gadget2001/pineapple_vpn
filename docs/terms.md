# Пользовательское соглашение Pineapple VPN

Дата: 2026-03-11

## 1. Общие положения
Pineapple VPN предоставляет сервис защищенного удаленного доступа к российскому IP для пользователей, находящихся за границей. Сервис не предназначен для обхода блокировок.

## 2. Предмет договора
Сервис предоставляет доступ к VPN-конфигурации на условиях подписки. Пользователь получает доступ к сервису и обязуется соблюдать правила использования.

## 3. Ограничения
Запрещается использовать сервис для незаконной деятельности. Пользователь несет ответственность за все действия, совершенные через сервис.

## 4. Подписка и оплата
Доступ предоставляется на платной основе согласно выбранному тарифу. Пробный период предоставляется автоматически при регистрации.

## 5. Ответственность
Провайдер сервиса не несет ответственности за действия пользователя и последствия использования сервиса. Пользователь обязуется соблюдать законодательство страны пребывания и РФ.

Сервис может изменять условия, публикуя обновления в интерфейсе и/или на сайте.
