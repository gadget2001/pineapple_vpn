# Публичная оферта и пользовательское соглашение Pineapple VPN

Версия: 2026-03-15  
Дата вступления в силу: 15.03.2026

## 1. Статус документа
Настоящий документ является официальным предложением (публичной офертой) заключить договор возмездного оказания услуг Pineapple VPN на условиях, изложенных ниже.

Акцепт оферты осуществляется пользователем путем совершения действий в интерфейсе сервиса, которые однозначно подтверждают согласие с условиями:
- подтверждение согласия с юридическими документами в onboarding;
- активация trial-периода;
- пополнение кошелька;
- покупка тарифа;
- получение и использование VPN-конфигурации при активном доступе.

С момента акцепта пользователь считается заключившим договор на условиях настоящей оферты.

## 2. Термины
Для целей настоящей оферты используются следующие термины:
- **Сервис** — Pineapple VPN, включая Telegram-бот, Telegram MiniApp и backend API.
- **Пользователь** — дееспособное физическое лицо, использующее сервис.
- **Trial (пробный период)** — ограниченный по сроку бесплатный доступ, предоставляемый на условиях сервиса.
- **Подписка** — платный период доступа к функциям сервиса.
- **Кошелек** — внутренний учетный баланс пользователя в рублях для оплаты тарифов.
- **Конфигурация** — персональная subscription-ссылка и связанные технические параметры для подключения в совместимом клиенте.

## 3. Предмет договора
Оператор предоставляет пользователю услугу защищенного удаленного доступа к российским сервисам из-за границы в рамках выбранного тарифа или активного пробного периода.

Сервис Pineapple VPN не позиционируется и не предоставляется как инструмент обхода блокировок.

## 4. Порядок подключения и использования
1. Пользователь авторизуется через Telegram.
2. Пользователь знакомится с юридическими документами и подтверждает согласие.
3. Пользователь проходит onboarding (в том числе выбор ОС и инструкцию).
4. После выполнения условий доступа пользователь получает конфигурацию.
5. Для продолжения использования сверх trial требуется активная подписка.

## 5. Trial-период
1. Trial предоставляется однократно для каждого пользователя.
2. Базовая длительность trial — 3 дня.
3. При входе по реферальной ссылке trial может быть увеличен (например, до 7 дней) в соответствии с актуальными настройками сервиса.
4. Trial активируется вручную пользователем через интерфейс; автоматическая активация не применяется.
5. Если пользователь уже использовал trial ранее, повторная активация не предоставляется.

## 6. Тарифы и стоимость
1. Актуальные тарифы и цены публикуются в интерфейсе сервиса.
2. Стоимость тарифов, длительность и условия могут изменяться оператором.
3. Изменение стоимости не влияет на уже оплаченный период доступа.
4. Новая стоимость применяется к последующим покупкам/продлениям.

## 7. Оплата и кошелек
1. Оплата доступа осуществляется через пополнение внутреннего кошелька.
2. Минимальная сумма пополнения определяется в интерфейсе и backend-валидацией.
3. Покупка тарифа производится списанием средств из кошелька.
4. При недостаточном балансе операция покупки тарифа не выполняется.
5. Платежи обрабатываются внешним платежным провайдером; оператор не хранит полные карточные данные пользователей.

## 8. Срок действия доступа и продление
1. Доступ действует в течение оплаченного периода либо trial.
2. При оплате до окончания текущего периода продление может выполняться от даты завершения действующего периода (без потери оставшихся дней), если это предусмотрено текущей логикой сервиса.
3. После истечения срока подписки доступ может быть ограничен, а VPN-профиль деактивирован.
4. При возобновлении подписки может использоваться тот же профиль/ключ в соответствии с архитектурой сервиса.

## 9. Реферальная программа
1. Пользователь может получать реферальную ссылку и приглашать новых пользователей.
2. Условия реферальной программы (размер бонуса, правила начисления, ограничения) определяются текущими правилами и интерфейсом сервиса.
3. Оператор вправе изменять условия программы в одностороннем порядке.
4. Начисления по операциям с признаками злоупотреблений могут быть приостановлены, пересчитаны или отменены.

Пользователь обязуется:
1. использовать сервис законно и добросовестно;
2. соблюдать документы сервиса (оферта, политика данных, правила использования);

1. пользоваться сервисом в пределах оплаченного доступа;
2. получать информацию о тарифах, платежах и статусе подписки;
